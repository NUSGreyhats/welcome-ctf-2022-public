# triple

### Challenge Details
A file has been triple-encrypted with three different keys

### Key Concepts
[XOR](https://en.wikipedia.org/wiki/Exclusive_or)

### Solution
XOR flag with `1`, `2`, and `3`.

### Learning Objectives
XOR is reversible, no matter how many times you do it
