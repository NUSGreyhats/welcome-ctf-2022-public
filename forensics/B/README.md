# B

### Challenge Details
A binary message is hidden after the gif's data

### Key Concepts
- GIF file format
- Binary can come in different forms

### Solution
1. Find the `»`s and `B`s in the file
2. Map `»` to 0, and `B` to 1, then decode as binary
